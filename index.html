📚 教程

🚀 快速教程
📖 详细教程

5 步快速上手

1
API：进入 API 填 Key → 点 🔄 拉模型 → 保存。
2
角色：进入 角色 导入 PNG / JSON 卡，或手动创建。
3
面具：进入 面具 创建你自己（{{user}} 替换成面具名）。
4
对话：进入 对话 点 ＋ 选角色 + 面具，开聊。
5
美化：对话框右上 ≡ 单独美化此对话；外观 App 管全局。

💡 小贴士

  • 对话页输入框旁的 ＋ 菜单可导出聊天记录。
  • 角色编辑里点开场白旁的 ＋/− 增减，每个开场白独立换行。
  • 预设可绑定正则一起导出，三档正则结构清晰：预设 / 独立 / 角色卡。
  • 世界书按「册」管理，每册可独立导出。
  • 「内存」页可一键导出/导入全部数据，跨设备搬家。

📋 预设（Preset）

预设是按顺序拼接成系统提示词的「条目」集合。

  • 新建册：点击「＋ 新建」可创建多个预设册，互相独立。
  • 条目：每条 name + role + content，开关控制是否注入。
  • 排序：拖动 ≡ 排序。
  • 捆绑正则：把独立正则绑到当前预设，导出时一起打包。
  • 导出：导出当前预设册（含条目+捆绑正则）为 JSON。
  • 占位符：{{char}} 替换角色名，{{user}} 替换面具名。

🔣 正则三档

  • 预设捆绑正则：跟当前预设绑定，切预设跟着切换。
  • 独立正则：始终启用，跨所有对话。
  • 角色卡正则：导入角色卡时自带，使用该角色对话时自动启用。

📖 世界书 + 🎴 角色卡

  • 世界书每册一个独立的小百科，每条有关键词和内容。
  • 角色卡可绑定世界书 + 多个独立正则，导出时一起打包。
  • 导入 PNG / JSON 角色卡自动带入开场白 / 世界书 / 正则。
  • 每个角色卡可设多个开场白（在角色编辑里 ＋/−），切换时自动分段。

✨ 美化

外观（全局）：主题、强调色、主页背景、状态栏、时间颜色、App 图标、全局字体、主页小组件。

对话框美化（单独）：对话页右上 ≡，每个角色独立保存。

主题里有「全透明」预设可叠在系统壁纸上。

💾 数据备份

「内存」App 顶部 📤 一键导出整个数据库 JSON，包含所有：API 配置、角色、面具、预设、世界书、正则、对话历史、CSS、字体、音乐元数据等。
